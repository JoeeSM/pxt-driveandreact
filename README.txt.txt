follow these steps to run the editor:
1. Nodejs package manager is required for this editor. link: https://nodejs.org/en/
2. Use the command: npm install -g pxt
3. Load into the 'pxt-driveandreact' directory with a command line interface.
4. Use the command: pxt serve (This is to build and run the local server).

If step 4 didn't work, please use the following command: npm install

The Microsoft MakeCode team have documentation on this process at: https://makecode.com/target-creation
